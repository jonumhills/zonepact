How It Works

Five steps.
Verifiable end-to-end.

Step 1

User types a query

"What is going on with 3902 Stratford Ct?" or a parcel PIN

Step 2

x402 micropayment

Wallet signs 0.001 ETH tx on Base Sepolia. Backend verifies on-chain before processing.

Step 3

Claude agent (Bedrock)

Calls lookup_parcel → Supabase. Calls get_onchain_history → Base Sepolia.

Step 4

On-chain verification

ZonePactRegistry returns zoning change + vote result + recorded_at timestamp. BaseScan link included.

Step 5

Report + Alert

S3 stores verified JSON report. SNS fires email alert. Map highlights parcel polygon.

Example query: "What is going on with 3902 Stratford Ct?" → Owner · PIN · Present zoning NB-2 → Proposed MU-2 · Petition Z-32-2023 · Approved 5-1 · ⛓ Verified on Base Sepolia
